# GrindMode Website Upload

Dieser Ordner ist der finale statische Upload-Ordner fuer die GrindMode-Webseite.

## Dateien zum Hochladen

- `index.html`
- `privacy.html`
- `terms.html`
- `imprint.html`
- `faq.html`
- `site.css`
- `site.js`
- `app-icon.png`

Alle Dateien muessen gemeinsam in dasselbe Web-Verzeichnis hochgeladen werden.

## Noch zu ersetzen

- Impressum: Name/Firma, Anschrift, E-Mail, ggf. Telefon, Vertretungsberechtigte Person, USt-ID.
- Datenschutz: echte Support-/Datenschutz-E-Mail.
- App-Store-Link: aktuell nicht eingebunden, weil kein echter Store-Link vorliegt.
- Screenshots: aktuell nutzt die Startseite ein CSS-App-Mockup. Echte Screenshots koennen spaeter ergaenzt werden.

## Datenschutz-Status der Website

- Keine Tracking-Skripte.
- Keine externen Fonts.
- Kein CDN.
- Kein Cookie-Banner notwendig, solange keine Tracking-/Marketing-/externen Medien nachgeruestet werden.

## App-Status laut Website

- Lokale Speicherung.
- Kein Konto.
- Kein Firebase.
- Keine aktive Werbung.
- Keine aktive Store-Abrechnung.
- Export/Import lokal per JSON.
